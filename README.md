# 🎬 Moive App 2019

</br>

> React JS를 사용한 반응형 영화 앱 </br></br>
매주 새로운 영화들이 업데이트됩니다. (영상은 제공 안 함)
</br>

## 1. 페이지
* Home </br>
20편의 영화들이 소개되어 있습니다.
* Detail </br>
영화에 대한 상세한 설명입니다.
* About </br>
웹 사이트에 대한 작자의 말입니다. </br>

</br>

## 2. 기술 스택
* React
* JSX
* Class Components
* Data Fetching
* Routing
* Axios

</br>
